KubeCon China 2019 Co-Located Event SOFAStack Cloud Native Workshop

2019-06-04 · Jimmy Song
SOFAStack Cloud Native Workshop hosted by Ant Financial (KubeCon China 2019 Co-Located Event) Date: Monday, 24 June, 2019 Time: 9:00 – 16:00 Location: Shanghai Expo Centre Room 616 Registration Fees: Complimentary Register here: https://www.lfasiallc.com/events/kubecon-cloudnativecon-china-2019/co-located-events/#sofastack-cloud-native-workshop Note: This event is hands-on, please bring your personal computer. The language of communication in this workshop is Chinese. SOFAStack (Scalable Open Financial Architecture Stack) is a financial-grade distributed architecture independently developed and open sourced by Ant Financial.
